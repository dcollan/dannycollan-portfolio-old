Charm Social Media App

Done Using Java and XML

After graduating SFSU in May 2021, I began to work independently on a mobile app with features alike Instagram, where I would be able to create an account, log into it, and then post photos, where these would be shown up in my profile page. Liking and commenting are also enabled for this app. For this app, I have used Java to link all of my functionalities to my database in Google Firebase, as well as granting the app capabilities to read any sort of entered information (such as the username or password), allowing sign-in with a Google account, creating an account, or to simply navigate to a different panel by clicking on an icon.


To do the design and layout of the mobile app, I have used XML classes. Just like my start-up mobile app, Homerun, I used this in a manner alike CSS where I designed how the login page, create an account page, home page, profile page, etc. would appear. The Java classes I have used would read from matching attributes found in those XML classes, where with a little coding, grant those attributes with an action as the user interacts with them.

Though the mobile app is still currently in progress (e.g. - I would still need to add the story functionality, as well as being able to delete posts from the app or to be able to log out of the account), this has yet been the biggest experience yet. Being able to program such a marvelous piece of work, where I am able to view the app on my android as if it were my own Instagram app provides me with high confidence that I am, indeed, truly determined with the kinds of work I do.
