MusicalWorld

Done Using HTML, CSS, JavaScript, & PHP

In this project, I decided to independently work on a website using PHP that allows for the user to register an account or sign into an existing one, where the person would be able to create a certain playlist by uploading any desired song from his/her OS.

The user is free to listen to any of their uploaded songs, and has the option to send a message to me for any inquiries by email or phone. The server can be run by entering "localhost/Musical-World/" in the web browser. For this case using WAMP, I used Microsoft Edge as my web browser to run my PHP files.

Though the website is supposed to serve as a simple web application that allows for the user to upload and listen to their desired songs through a playlist, its main purpose is to give them a unique and satisfying user experience. By implementing a modern design to each page with an excellent use of HTML, CSS, and JavaScript, it will ensure the user that they are inside a highly trustworthy and professional website.
